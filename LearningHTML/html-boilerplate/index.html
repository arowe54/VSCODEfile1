Notes:

index.html is the homepage. !DOCTYPE html gets HTML5 (latest version).

html is the root element. lang specifies language (en = english). meta charset="utf-8" makes sure website displays special symbols correctly.

Head gives important meta-information about the webpage plus stuff to render correctly in the browser. Don't put elements that display content in head.

Body holds all content displayed to users. Press ! and first option for a boiler plate shortcut.
